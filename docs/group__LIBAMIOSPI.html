◆ SPI_getMaxBufSize()

ssize_t SPI_getMaxBufSize ( )

Gets the maximum buffer size for the SPI driver.

Note
This function reads from the filesystem, so you should avoid calling it more than once, when possible. If this function fails, the caller should assume the size is 4096, as that's the default out of the box.
Returns
The size of the buffer, or -1 on error and sets errno.
